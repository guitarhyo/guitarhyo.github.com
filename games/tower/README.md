## 학습 포인트

### 공중 액션 시스템

적 처치 시 캐릭터가 반동으로 다시 상승하도록 설계. `enemyHitBounce()` 구조를 통해 공중 체공 유지 → 연속 콤보 → 빠른 템포 액션을 구현한다.

콤보 유지 시간 내 적을 연속 처치하면 추가 점수 및 이펙트 강화.

### 이동 및 점프 물리

중력(`gravity`)과 속도(`velocity`)를 직접 계산하는 방식 사용.

```js
player.vy += gravity;
player.y += player.vy;
```

공중 대시 사용 시 속도를 즉시 변경하여 빠른 이동 연출 구현.

### 모바일 입력 통합

Pointer Events(`pointerdown` / `pointerup` / `pointermove`) 기반 입력 처리로 터치·마우스·모바일 브라우저를 단일 로직으로 대응.

```js
canvas.setPointerCapture(e.pointerId); // 화면 밖으로 손이 벗어나도 입력 유지
```

### 캐릭터 시스템

캐릭터별로 이동 속도, 공격 스타일, 스킬, 이펙트를 다르게 설계.

| 캐릭터 | 특성 |
|--------|------|
| 🐱 냥이 검사 | 빠른 공중 콤보 특화 |
| 🐶 멍멍이 검사 | 범위형 묵직한 공격 특화 |

### 도트 애니메이션

16×16 또는 24×24 스프라이트 기준 제작. 프레임 기반 애니메이션 구조 사용.

```js
frameIndex = (frameIndex + 1) % maxFrame;
```

작은 도트에서도 타격감·반동·속도감이 느껴지도록 연출.

### 스킬 이펙트

캐릭터보다 큰 이펙트를 사용하여 화려한 액션 강조.

- 검기, 픽셀 폭발, 잔상, 대시 흔들림
- Canvas의 `globalAlpha`, `screen shake`, `particle` 구조 활용

### 무한 등반 구조

카메라는 플레이어 위치를 따라 위로 이동. 플레이어가 일정 높이에 도달하면 새로운 적·발판·배경을 생성하여 무한 등반 구조 구현.

### 점수 및 저장

점수는 **높이 + 콤보 + 처치 수** 기준으로 계산. 최고 기록은 `localStorage`에 저장.
